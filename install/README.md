### Vim in Ubuntu 14.04

Save following into gvim.desktop under ~/.local/share/applications 

> [Desktop Entry]
> Categories=;
> Encoding=UTF-8
> Type=Application
> Name=GVIM
> Icon=gvim.png
> Exec=gvim %f
> StartupNotify=false
> StartupWMClass=Gvim
> OnlyShowIn=Unity;TextEditor;
> X-UnityGenerated=true
> Version=1.0
> MimeType=text/plain;
